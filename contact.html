Mastodon

I tried it, but found it was still a bit social-media-y for what I wanted. I’ll probably write a review at some point but whilst I found the technical side of the ActivityPub protocol interesting, I found the implementation to require a bit too much work for little benefit to me. It seems to work for a lot of people though so don’t write it off because one person says this!
